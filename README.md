# TITAN BLINKER
Custom LED blinker with mechanical switches built using the XIAO RP2040.

___________________________________________________________________________________________________________________________________________________________________________

## Overview

Titan Blinker is LED blinker built using XIAO RP2040 microcontroller and mechanical switches. The Titan blinker contains electronics,pcb designing and firmware designing.

Unlike the other LED blinker Titan blinker contains XIAO RP2040 and switches. Most of the blinker uses battery instead of microcontroller but titan blinker uses XIAO RP2040 and switches programmed using Arduino. The switches helps in turning on the different LED's independently. The use of switch trigger different lighting behaviour

These project is built using XIAO RP2040,cherry MX button switch, DSA key caps,3 resistors and LEDs.

___________________________________________________________________________________________________________________________________________________________________________

## Inspiration

The idea of creating Titan Blinker came from idea of creating a compact and switch controlled LED blinker.I wanted to make the project different thats why I used cherry MX switch which provides tactical feedback and feels awesome and I don't want LEDs to blink continuously like normal blinkers. The use of the MX cherry allows turning on and off of LEDs using switches which seems to be different.

My project is inspired by the project found on the stasis hackclub made by the Meghana Madiraju.
